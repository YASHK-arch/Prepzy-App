Problem Description

You are given an array A of integers of size N.

Your task is to find the equilibrium index of the given array.

The equilibrium index of an array is an index such that the sum of elements at lower indexes is equal to the sum of elements at higher indexes.

Note:

  • Array indexing starts from 0.
  • If there is no equilibrium index then return -1.
  • If there are more than one equilibrium indexes then return the minimum index.
  • If there are no elements at lower indexes or higher indexes, then the corresponding sum is considered as 0.

Problem Constraints

1 ≤ N ≤ 10⁵

-10⁵ ≤ A[i] ≤ 10⁵

Input Format

First argument is an array A.

Output Format

Return the equilibrium index of the given array. If no such index is found then return -1.

Example Input

Input 1: A = [-7, 1, 5, 2, -4, 3, 0]
Input 2: A = [1, 2, 3]

Example Output

Output 1: 3
Output 2: -1

Example Explanation

Explanation 1:

i=3 is an equilibrium index, because:
A[0] + A[1] + A[2] = A[4] + A[5] + A[6]

Explanation 2:

There is no such index. 
            
public class Equilibrium_index_of_an_array {
    public int solve(int[] A) {
        int n = A.length;
        // Using Prefix Sum and Suffix Sum logic
        int [] prefix = new int [n+1];
        for(int i = 0; i < n; i++){
            prefix[i+1] = prefix[i] + A[i];
        }
        
        int [] suffix = new int [n+1];
        for(int i = n-1; i >= 0; i--){
              suffix[i] = suffix[i+1] + A[i];
        }
        
        int min_index = Integer.MAX_VALUE;
        boolean exist = false;
        
        // Find the index where sum of lower indices == sum of higher indices
        for(int i = 0; i < n; i++){
            // sum before i (prefix[i]) == sum after i (suffix[i+1])
            if(prefix[i] == suffix[i+1]){    
                min_index = Math.min(min_index, i);
                exist = true;
            }
        }
        
        if(exist == false){
            return -1;
        }
         
        return min_index;
    }
}
