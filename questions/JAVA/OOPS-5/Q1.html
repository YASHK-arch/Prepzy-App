Problem Description

Design a data structure that supports adding new words and searching for words with optional wildcard characters.

Implement the following functions:

  • WordDictionary() — Initializes the object.
  • void addWord(word) — Adds a word (containing lowercase English letters) to the structure.
  • bool search(word) — Returns true if there is any word in the structure that matches word; word may contain dots '.', where each dot can represent any single letter.

Problem Constraints

1 ≤ word.length ≤ 25

Words in addWord contain only lowercase English letters.

Words in search may contain lowercase letters and dots '.'.

There will be at most 2 dots in word for search queries.

At most 10⁴ calls will be made to addWord and search.

Input Format

You need to complete the functions alone. Do not print anything.

For custom testing, the input will be given as:

Q
add <word>
search <pattern>
...
  • The first line contains the number of function calls Q.
  • Each of the next Q lines represents one function call:
    • add <word> → Adds a word to the data structure.
    • search <pattern> → Returns true or false.

Output Format

For each search function call, return true if the word is found, otherwise false.

Example Input

Input 1:

["WordDictionary","addWord","addWord","addWord","search","search","search", "search"]
[[],["bad"],["dad"],["mad"],["pad"],["bad"],[".ad"],["b.."]]

Input 2:

["WordDictionary","addWord","addWord","search","search","search"]
[[],["cat"],["car"],["cat"],["c.t"],["bat"]]

Example Output

Output 1:

[null,null,null,null,false,true,true,true]

Output 2:

[null,null,null,true,true,false]

Example Explanation

Explanation 1:

WordDictionary wordDictionary = new WordDictionary();
wordDictionary.addWord("bad");
wordDictionary.addWord("dad");
wordDictionary.addWord("mad");
wordDictionary.search("pad"); // return False
wordDictionary.search("bad"); // return True
wordDictionary.search(".ad"); // return True
wordDictionary.search("b.."); // return True 
            
import java.util.ArrayList;

class WordDictionary {
    private ArrayList dictionary;

    public WordDictionary() {
        dictionary = new ArrayList<>();
    }

    public void addWord(String word) {
        dictionary.add(word);
    }

    public boolean search(String word) {

        // Case 1: word has NO wildcard
        if (!word.contains(".")) {
            return dictionary.contains(word);
        }

        // Case 2: word HAS wildcard
        for (String dictWord : dictionary) {

            // length must match
            if (dictWord.length() != word.length()) {
                continue;
            }

            boolean match = true;

            // match character-by-character
            for (int i = 0; i < word.length(); i++) {
                char wc = word.charAt(i);       // char in wildcard pattern
                char dc = dictWord.charAt(i);   // char in dictionary word

                if (wc != '.' && wc != dc) {
                    match = false;
                    break;
                }
            }

            if (match) return true;
        }

        return false;
    }
}
